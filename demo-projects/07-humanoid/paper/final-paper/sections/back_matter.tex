\section*{Appendix}

The following figure represents a user prompt that converted the v0.6.0 draft paper into the first full generated AI paper. The prompt included instructions to use specific directories, files, and instructions to populate the draft template with the majority of the text represented in this final paper. The files were uploaded directly from Claude Code to GitHub based on efficient 12 commit instructions to reduce LLM working memory issues. A 13th repository commit was also specified in the instructions to fix any AI errors. Lastly, a 14th commit was submitted that updated relevant documentation throughout Clinical-AI-Demos.
